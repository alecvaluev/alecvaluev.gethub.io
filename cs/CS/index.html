# Introduction

VuePress is composed of two parts: a minimalistic static site generatoropen in new window with a Vue-powered theming systemopen in new window and Plugin APIopen in new window, and a default themeopen in new window optimized for writing technical documentation. It was created to support the documentation needs of Vue's own sub projects.

Each page generated by VuePress has its own pre-rendered static HTML, providing great loading performance and is SEO-friendly. Once the page is loaded, however, Vue takes over the static content and turns it into a full Single-Page Application (SPA). Additional pages are fetched on demand as the user navigates around the site.

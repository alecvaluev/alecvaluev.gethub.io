# Reading Rules

1. const + `base` -> `double`
2. ㅎ <-> `base`  -> `strong`
3. ㅌ + *이* -> 치

   ㄷ + *히* -> 치

   ㄷ + *이* -> 지
   
   > ㅣ/ㅑ/ㅕ/ㅖ/ㅒ/ㅛ/ㅠ 
4. ㄹ + `base` -> `double`
5. `ㅂ/ㅍ` + ㄴ/ㅁ -> `ㅁ`
6. `ㅅ` + ㅣ/ㅑ/ㅕ/ㅖ/ㅒ/ㅛ/ㅠ -> `sh`
7. ㅂ + `ㄹ` -> ㅁ + `ㄴ`
8. ㄱ + `ㄹ` -> ㅇ + `ㄴ`
9. `ㄱ/ㄲ/ㅋ` + ㄴ/ㅁ -> `ㅇ`
10. ㅇ/ㅁ + `ㄹ` -> `ㄴ`
11. `ㄴ` + ㄹ **or** ㄹ + `ㄴ` -> `ㄹ`
